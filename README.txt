Mundo Smart TV - sitio listo
---------------------------

Archivos:
- index.html
- catalogo.html (conectado a Google Sheets)
- styles.css

Pasos rápidos:
1) Abre la Google Sheet y en Archivo → Publicar en la web → selecciona la hoja y publica.
2) Sube estos archivos a GitHub (repo) y activa GitHub Pages o deploy en Netlify/Vercel.
3) Verifica que la hoja esté publicada: la URL para CSV ya está configurada con el SHEET_ID 1-XvTphECWH1qlJvIzMAHrhRYyFURMDoF.
4) Actualiza precios/stock pegando en la hoja; la página leerá los cambios al recargar o al usar Refrescar.
